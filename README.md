## Documentación de API
**Swagger Configurado y Funcional**  
Hemos implementado y validado `SwaggerConfig` para:
- Generar documentación automática de la API REST en tiempo real
- Probar endpoints directamente desde la interfaz interactiva
- Visualizar parámetros, ejemplos de request/response y códigos de estado
- Documentar esquemas de autenticación (JWT, API Key, etc.) 


### Acceso rápido
Al ejecutar la aplicación, accede a la documentación en:  
`http://localhost:8080/swagger-ui.html`
